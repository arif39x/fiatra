# 7-Day Blending Plane

## Muse ⨯ Ursina → Aetherium Studio

This folder is a **strategic analysis + build blueprint**. It dissects two existing projects (Muse + Ursina), extracts their core DNA, and blends them into something new — not a copy of either, but a **modern-era alternative** that supersedes both.

### Why Blend?

| | Muse | Ursina |
|---|---|---|
| **Render** | Rust/WGPU (fast, modern) | Python/Panda3D (legacy) |
| **API** | Heavy, character-focused | Light, entity-focused |
| **LLM** | Native, integrated | Nonexistent |
| **Manual** | Minimal (chat only) | Rich (prefabs, editor) |
| **Scene** | Single character | Full scene graph |
| **Primitives** | None | Dozens |

Each has what the other lacks. The blend is a **zero-compromise** tool.
